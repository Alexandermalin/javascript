# 表达式和语句
- 表达式
表达式UI坤车被求职的代码,JavaScript引擎会将其计算出一个结果.
  > x = 7
  > 3 + 4
  > num ++

- 语句
语句以一段可以执行的代码.
比如:prompt() 可以弹出一个输入框 还有if语句 for 循环语句等等

**区别:**
表达式:因为表达式可以被求值,所以它可以写在赋值语句的右侧 
- 表达式 num = 3 + 4 

语句:而语句不一定有值,所以比如alert() for  和break 等语句就不能被用于赋值
- 语句 alert()弹出对话框 console.log()控制台打印输出
